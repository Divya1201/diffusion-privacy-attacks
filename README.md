# diffusion-privacy-attacks

Implementing memorization attacks for newly generated images from diffusion models.

## Project setup

This repository is scaffolded as a Python project for diffusion model experiments with these dependencies:

- PyTorch
- diffusers (Hugging Face)
- numpy
- torchvision
- Pillow (PIL)

Install dependencies:

```bash
pip install -e .
```

## Folder structure

- `models/` — model checkpoints and weights
- `data/` — datasets and intermediate data
- `outputs/` — generated outputs and artifacts
- `utils/` — utility Python helpers
